# 题库优化方案

## 一、现有问题诊断

### 1. 维度分布失衡
- **商业服务**：几乎每道题都有2-3分，导致80%的人测出商业精英
- **医疗健康**：分数偏低，医学专业学生测不出医疗岗位
- **其他维度**：分布不均，缺少针对性题目

### 2. 题目同质化严重
- 大量题目都在问"你喜欢什么"，缺少"你能做什么"
- 缺少极端场景题，区分度不够
- 缺少大学生视角的题目

### 3. 专业门槛考虑不足
- 专业背景只是一个标签，没有在后续题目中强化
- 高门槛专业（医学、工程、教育）的学生容易被误判

## 二、优化策略

### 策略1：调整现有题目分数权重

**原则：**
- 降低"商业服务"的普遍性得分（从2-3分降到1-2分）
- 提高专业相关维度的得分（医疗、工程、科研等）
- 增加极端选项的分数差异（从1-3分拉大到1-5分）

**具体调整：**

#### Q4: 团队项目中，你更喜欢负责什么？
- A. 协调团队、分配任务、推动进度
  - 原：商业服务+3, 自主创业+2
  - 改：商业服务+2, 自主创业+2
  
- B. 和客户沟通、了解需求、收集反馈
  - 原：商业服务+2, 教育培训+2
  - 改：商业服务+1, 教育培训+2, 医疗健康+1

#### Q7: 你更看重工作的什么？
- A. 能发挥专业技能，成为某个领域的专家
  - 原：科研创新+3, 医疗健康+2, 工程制造+2
  - 改：科研创新+4, 医疗健康+4, 工程制造+3
  
- E. 能获得高收入和社会地位
  - 原：商业服务+2, 自主创业+2
  - 改：商业服务+3, 自主创业+2

#### Q12: 你对工作收入的期望？
- A. 收入很重要，希望获得高薪
  - 原：商业服务+3, 自主创业+2
  - 改：商业服务+3, 自主创业+3
  
- C. 收入不是最重要的，更看重工作意义
  - 原：医疗健康+2, 教育培训+2, 公共服务+3
  - 改：医疗健康+4, 教育培训+4, 公共服务+4

### 策略3：实现专业加权逻辑

在测评引擎中增加专业加权，确保专业背景对结果有显著影响。

**加权规则：**

| 专业 | 加权维度 | 加权系数 | 触发条件 |
|------|---------|---------|---------|
| 医学/护理 | 医疗健康 | 1.5x | 基础分>30 |
| 计算机/软件 | 科研创新 | 1.3x | 基础分>40 |
| 工程/制造 | 工程制造 | 1.4x | 基础分>35 |
| 商科/管理 | 商业服务 | 1.2x | 基础分>45 |
| 艺术/设计 | 文化艺术 | 1.4x | 基础分>40 |
| 教育/心理学 | 教育培训 | 1.4x | 基础分>35 |
| 理工科 | 科研创新 | 1.3x | 基础分>40 |
| 文科 | 文化艺术 | 1.2x | 基础分>35 |

**实现逻辑：**
```typescript
// 在 calculateScores 方法后增加专业加权
applyMajorWeighting(scores: DimensionScores, userMajors: MajorTag[]): DimensionScores {
  const weightedScores = { ...scores };
  
  userMajors.forEach(major => {
    switch(major) {
      case 'major_medical':
        if (scores['医疗健康'] > 30) {
          weightedScores['医疗健康'] = Math.min(100, scores['医疗健康'] * 1.5);
        }
        break;
      case 'major_cs':
        if (scores['科研创新'] > 40) {
          weightedScores['科研创新'] = Math.min(100, scores['科研创新'] * 1.3);
        }
        break;
      case 'major_engineering':
        if (scores['工程制造'] > 35) {
          weightedScores['工程制造'] = Math.min(100, scores['工程制造'] * 1.4);
        }
        break;
      case 'major_business':
        if (scores['商业服务'] > 45) {
          weightedScores['商业服务'] = Math.min(100, scores['商业服务'] * 1.2);
        }
        break;
      case 'major_art':
        if (scores['文化艺术'] > 40) {
          weightedScores['文化艺术'] = Math.min(100, scores['文化艺术'] * 1.4);
        }
        break;
      case 'major_education':
        if (scores['教育培训'] > 35) {
          weightedScores['教育培训'] = Math.min(100, scores['教育培训'] * 1.4);
        }
        break;
      case 'major_science':
        if (scores['科研创新'] > 40) {
          weightedScores['科研创新'] = Math.min(100, scores['科研创新'] * 1.3);
        }
        break;
      case 'major_liberal_arts':
        if (scores['文化艺术'] > 35) {
          weightedScores['文化艺术'] = Math.min(100, scores['文化艺术'] * 1.2);
        }
        break;
    }
  });
  
  return weightedScores;
}
```

## 三、预期效果

优化后，预期达到：
1. **维度分布更均衡**：商业精英占比从80%降到25%左右
2. **专业匹配更准确**：医学专业学生80%以上匹配到医疗岗位
3. **千人千面**：相同专业的学生，根据答题差异，能匹配到不同岗位
4. **区分度更高**：极端选项的分数差异从1-3分拉大到1-5分
